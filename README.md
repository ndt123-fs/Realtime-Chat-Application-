# ChatRealtimePlatform

A realtime chat application built with Spring Boot and WebSocket,
supporting instant messaging, JWT authentication, and basic video calling.

## Tech Stack
- Backend: Java, Spring Boot, WebSocket (SockJS, STOMP), JWT, REST API, MySQL
- Frontend: JavaScript, React

## Features
- Realtime messaging between multiple users
- Online/offline user presence tracking
- JWT-based authentication
- RESTful APIs for user and message management
- Basic video call functionality
